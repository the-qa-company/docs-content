You can already ask the first questions on top of your QA system, either by selecting it from the **QA System** page by clicking on the magnifying glass icon, which will redirect you to the **Ask** page.

<Image src='/img/newUI/cocktails/qa-system-page-cocktails.png' alt='home page'/>

You can also access it from directly from the **Ask** page by selectionning it using the top right selector.

<Image src='/img/newUI/cocktails/select-qasystem-ask.png' alt='ask page'/>

For the "Cocktails" QA system you could ask for example: "What is a Margarita?" or "What are the ingredients of Margarita?":

<Image src='/img/newUI/cocktails/cocktails-margarita.png' alt='margarita'/>

<Image src='/img/newUI/cocktails/margarita-ingredients.png' alt='margarita ingredients'/>

Note that for now there is no contextual information attached to the answers, like labels, descriptions, external links, maps, images and videos. How this can be achieved
will be shown in part [Contextual information](doc1.6).

The dataset contains some popular cocktails like: Margarita, Sex on the Beach, Long Island Iced Tea and Grasshopper.

The results that you achieve may not be satisfying. To train the machine learning model to fit your dataset check the next section.
