You can use as a data source a webpage or an entire website. In this section we describe how to use a website as a data source.

:::info
Note that we support both server and client side rendered web pages!
:::

Click on "Website" to add a web page or a website as a data source:

<Image src='/img/screenshots/tutorial-ui/data-source/data-source-empty.png' alt='data source menu'/>

You will get to the following page:

<Image src='/img/screenshots/tutorial-ui/data-source/website/website-empty.png' alt='Empty wizard'/>

## Add manually selected URLs

Enter the URLs of the pages you want to add and click on *Add Page* or press *Enter*. Your URLs will be appearing in the list below:

<Tabs>
    <TabItem value='input' label='Input'>
        <Image src='/img/screenshots/tutorial-ui/data-source/website/website-first-input.png' alt='Add first link'/>
    </TabItem>
    <TabItem value='added' label='Added' default>
        <Image src='/img/screenshots/tutorial-ui/data-source/website/website-first-add.png' alt='Add first link'/>
    </TabItem>
</Tabs>

## Crawl an entire website

Enter the URL a website you want to add and click on *Add Page and Subpages*. You will extract all the sub-pages under the particular URL.
You now have a list of all URLs under the website you specified which might be grouped.

<Tabs>
    <TabItem value='g1' label='Group sub-pages'>
        <Image src='/img/screenshots/tutorial-ui/data-source/website/website-after-extract.png' alt='Grouped sub-pages'/>
    </TabItem>
    <TabItem value='s1' label='Show sub-pages' default>
        <Image src='/img/screenshots/tutorial-ui/data-source/website/website-show-extracted.png' alt='Show sub-pages'/>
    </TabItem>
</Tabs>

:::info
You may see a<b style={{ verticalAlign: 'middle', color: '#ff4b4bb7', borderRadius: '50%', border: '1px solid #ff4b4bb7', padding: '2px 8px', margin: '0 10px' }}>i</b> symbol. By default the extraction skips link that do not contain the original url since this might be pages from another website. You can click on symbol to select additional urls you want to add as a data source.
<Image src='/img/screenshots/tutorial-ui/data-source/website/website-show-skippedlinks.png' alt='Show skipped-links'/>
:::

### Delete sub-pages

Some sub-pages might not interest you, you can remove them by clicking on the delete button (in the *Action* column).

<Image src='/img/screenshots/tutorial-ui/data-source/website/website-remove.png' alt='Remove'/>

### Move sub-pages

Your sub-pages are originally grouped but you can move them **out of their group** by clicking on the *Move* button (in the *Action* column).

## Advanced Settings (optional)

You can customize the website crwaling parameters if needed (default parameters are good for general use). This include for example:

- the deepth until which the crawling will be performed
- the maximal time the url extraction will be executed
- cookies or tokens to crawl a secured website or web application

<Image src='/img/screenshots/tutorial-ui/data-source/website/creation-settings.png' alt='Settings'/>
