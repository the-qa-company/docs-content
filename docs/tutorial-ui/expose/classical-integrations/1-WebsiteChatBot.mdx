## Customize
On the website chatbot you have access to several options :

You can customize the appearance of the widget :
* The initial message
* The theme
* If it's open or closed at the loading of the website
* Its position on the screenshots
* Its accent color
* Its CSS font property

You have an overview of the result on the right.

<Image src='/img/screenshots/tutorial-ui/expose/website-chatbot/website-chatbot1.png' alt='website chatbot'/>

## Access

You can choose your access type, you decide whether it is available to everyone who visits your website or only to authenticated visitors. And you can generate the API key for the integration of the widget.
