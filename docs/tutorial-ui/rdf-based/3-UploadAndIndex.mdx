## Creation
Pick the name and language of your QA System, upload your dataset (click or drag and drop on the dedicated area), you can click on:
* <kbd>Settings</kbd> to personnalize them (default parameters are good for general use)
* <kbd>Finish</kbd> to create your QA System (and use default parameters)

<Image src='/img/screenshots/tutorial-ui/rdf-based/creation-init.png' alt='Wizard'/>

You can also modify the list of properties that are used to assign names to the nodes. 

<Image src='/img/screenshots/tutorial-ui/rdf-based/creation-settings.png' alt='Wizard'/>

:::caution
The dataset can be in one of the common RDF serializations and must be syntactically correct. You can both check and clean a file using [serdi](https://drobilla.net/software/serd)

```
serdi -i ntriples -o ntriples dump.nt > dump_clean.nt
```
:::


After uploading the dataset, your QA system will appear in your home page.

It is now ready to be [queried!](query)
